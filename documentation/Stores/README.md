title: Data Stores
---

Polypheny currently supports the following data stores:

* Cassandra
* Cottontail-DB
* File
* HSQLDB
* MonetDB
* [MongoDB](Mongo.md)
* PostgreSQL

Data stores can be added using one of the following approaches. Not all approaches are supported by all adapters.

* Embedded
* Remote
* [Docker](Docker.md)
