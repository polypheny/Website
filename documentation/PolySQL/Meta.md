The META function allows to inspect data stored in columns of type `IMAGE`, `SOUND` and `VIDEO`.

{% highlight sql %}
META( col[, dirName[, tagName]] )  
{% endhighlight %}

Returns metadata information about a multimedia object. The information consists of metadata _tags_ that are grouped within _directories_.

The mandatory `col` parameter must be a column of a multimedia data type. If the `dirName` parameter is given, only tags of a specific directory will be returned. If the `tagName` is given, only the specific tag value will be returned.
